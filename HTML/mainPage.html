    Information is Power

    An application that intends to provide a well-established web-based social network system between students and a recruiter. Students will login into the system and see different opportunities based on their faculty. On this site there will be a lot of posts categorised in Opportunities, News, Universities, Places and Others. When creating a post the placement officer will have to select the category and related field of the post and the post will appear to students if and only if it matches their faculty. There are three roles in this system which are user, admin and super admin which have different access to the system. On each post a student will have an option to take an aptitude test which matches his faculty and the results are collected by the admin who can be able to invite a student to an interview if he is satisfied with the results.
